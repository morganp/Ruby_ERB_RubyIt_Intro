Arrays are Dynamic

but not sparse

Hashes

a_hash = {:one => 1, :two => 2}
puts a_hash[:one]
> 1

a_hash[:one] = 3
puts a_hash[:one]
> 3

a_hash[:three] = 1
puts a_hash
> {:one=>3, :two=>2, :three=>1}

Variable Scope (1)

Local Variable (no sigil)

no sigil

local_variable = 0

Instance Variable (@)

Available within Class, each instance is unique.

@instance_variable = 0

Variable Scope (2)

Class Variable (@@)

Best Avoided. Available to all instances of Class, including inheritance.
Only one version exisits.

@@class_variable = 0

Global ($)

Available to all, one version exists.

$global_variable = 0
